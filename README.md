## 📦 Getting Started

### Prerequisites
- Node.js 20+
- NPM 10+
- A Google AI Studio API Key

### Installation
1. Clone the repository: `git clone https://github.com/your-org/aegis-omega.git`
2. Install dependencies: `npm install`
3. Configure your key: Create a `.env` file with `GEMINI_API_KEY=your_key_here`
4. Deploy local substrate: `npm run dev`
